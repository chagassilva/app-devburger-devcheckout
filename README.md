🛒 Sobre o DevCheckout
O DevCheckout é uma aplicação web simples e funcional voltada para simular uma experiência de página de checkout. Ideal para projetos de e-commerce, o sistema permite ao usuário revisar seus produtos no carrinho e aplicar cupons de desconto de forma prática e intuitiva.

---

⚙️ Funcionalidades

Cadastro de dados do usuário ao realizar o pedido

Campo para de cumpom de desconto
